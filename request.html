The Request

The request carries data between layers of the application. The following are its properties:

  • event - the original DOM event that triggered the request, in jQuery form
  • view - the view which triggered the request
  • options - the options object that was used to initialize the request. It comes from the listener definition, and includes any defaults.
  • data - data coming from the server gets placed here for communication between both chained controller actions and controller actions and subscribed views.
  • target - the DOM element whose listener triggered the current request, as a jQuery object.
  • Plus any "_id" properties set by auto-id (check bottom of the section on the controller)

Stopping Execution

If at any point before or between ajax calls you need to stop the normal flow of the request, simply throw "halt":

   throw 'halt'

You can do this in either views or controllers
